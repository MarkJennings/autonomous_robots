### Duplicate the Repo
1. Make sure you're logged into your GitHub account.
2. Create a new repo with the same name under your GitHub account. Don't initialize that with anything.
3. `git clone <this repository url>` (found in the upper right)
4. `cd <cloned_repo>`
5. `git push --mirror <your new repository url>`

### Clone and Build
1. `mkdir projects`
1. `cd projects`
1. `git clone <your repository url>` (found in the upper right)
1. `cd <cloned_repo>`
1. `make -j`
